What is the release status of the tool?

HRSB is currenly in alpha. The are a load of features yet to be added and I'm sure a ton of bugs to be fixed.

What browsers are supported?

At this time, I've tested it in Chrome, Firefox and Edge(has some visual glitches). Chrome is recommended as its the primary development broswer and therefore has been tested most extensivly. Chrome also seems to have the best performace when parsing the replay

Why is feature 'X' not included

That depends, either I havent gotten around to it, noone has suggested it, or I simply can't get the data I need out of the replay. If you have suggestions on how to improve the tool feel free to post it here

Do you store replays on the server?

No. The app does everthing in the browser, replays are never uploaded to a server.

If you don't store anything on the server then how do things like saving the stat filters work?

The app uses local storage to store settings like the filters. If you move to another computer/browser the settings won't carry over

Do you collect any information about me or my replays?

The app uses Google analytics to gather anonamized statistcs about the apps usage. Non of the data identifies you. The only info about the replay that is gathered is the game mode, map and fingerprint of the replay

Why do i get blank pages on older replays

Some features use data that isn't included in older replays. Usually you will get a not supported message, but due to very little documentation about when replays start supporting certain data not all cases have been accounted for. If you run into this please post a bug report at https://github.com/myrddraall/heroesbrowser-replay-viewer/issues and provide a link to the replay

Why do I get a 'Failed to load Heroes Protocol' Error

The app needs data about the replay version in order to parse it. It get this data from Blizzard's heroprotocol project. If there has been a patch recently the protocol may just not be released yet. Please try again later. Blizzard usually releases protocols fairly quicklly. If the replay is older and you get this message please post a bug report at https://github.com/myrddraall/heroesbrowser-replay-viewer/issues and provide a link to the replay
